6. Элементы react
React элемент представляет собой объект (обычный JS-объект). Обязательно хранит в себе тип и свойства. props - св-ва элемента (в которых записано их внутренее содержимое). Создать react элементы можно с помощью функций h1, ol, li и т.д.(более простой и быстрый способ), с помощью react компонентов (более сложные react элементы) или с помощью функциональных компонентов (быстрый, но с ограниченными возможностями).
