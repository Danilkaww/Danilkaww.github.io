Жизненный цикл компонентов react При монтировании компонента: Вызывается конструктор -> происходит отрисовка -> вмонтирование компонента При обновлении компонента: Просматриваются новые свойства/состояние -> происходит отрисовка -> вмонтирование компонента При размонтировании: -> размонтирование/удаление компонента Группа функций Did (react вызывает когда уже сделано действие) componentDidMount - вызывается когда нужно перехватить состояние, компонент уже вмонтирован, состояние инициализировано, компонент прорисован. componentDidCatch - вызывается когда произошла ошибка в компоненте componentDidUpdate - вызывается когда компонент перерисовался (обновился) Группа функций Will (react вызывает до действия) componentWillReceiveProps - вызывается до получения свойств компонента componentWillUnmount - вызывается перед удалением компонента (например для сохранения данных из компонента) componentWillUpdate - вызывается до прорисовки компонента componentWillMount - функция, которая вызывается react`ом если компонент уже готов к монтированию (т.е. свойства уже инициализированы) shouldComponentUndate - вызывается для проверки надобности перерисовки компонента (функция для оптимизации)
