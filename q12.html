Параметры пути в react route navLink - функция, которая создает ссылку react route, имеет 2 параметра: путь, который нужно отобразить и текст, который мы указываем на ссылке Компонент switch - переключает вид приложения в зависимости от того, какой путь установлен с помощью ссылок navLink. Route - функция, которая прописывает пути, имеет несколько параметров. Основные: path - путь, при котором следует отрисовывать заданную часть и render - функция, которая рисует то, что нужно нарисовать при заданном пут и, exact - свойство, которое позволяет сравнивать пути, т.е. информация по пути /num будет выводиться, а по пути /num/house уже не будет (проверяет заданный путь и текущий путь вплоть до буквы). : - символ, после которого записывается параметр пути в URL. legacy context - возможность в react`е передавать набор данных от верхнего компонента во всем низлежащие компоненты, не записывая каждый раз в props.
