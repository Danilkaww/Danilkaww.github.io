8. Состояние компонентов react
Состояние компонента может изменяться от действия пользователя (клик), либо от внешних факторов. При смене состояния компонент меняется и перерисовывается, тем самым приложение получается интерактивным. Функция setState - функция react, приводит к активации функции render (перерисовке приложения). Функция для установки состояния apply, для изменения setState, обращаться можно без функций. Подъем состояния - информация касающаяся компонентов более низкого уровня пересятся в состояние компонента более высокого уровня. При подъеме состояния, оно передается в нижний компонент в качестве свойства.
