20. Функция connect в react-redux
Connect - функция для создания контейнеров, RConnect - определяет типы для всех данных, которые учавствуют в стыковке. Имеет следующие параметры:
-Состояние
-Аргумент функции dispatch
-Результат функции dispatch
-Собственные свойства контейнера
-Свойства получаемые из State
-Свойства, являющиеся функциями от dispatch
-Свойства оборачиваемого контейнера
Чтобы функция заработала нужно подключить Provider - функция, которая позволяет всем компонентам использовать хранилище
