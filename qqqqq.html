22. Библиотека kotlinx.serialization
Библиотека, которая позволяет передавать данные между сервером и клиентом в удобном формате. Серелизация - это процесс представления объекта в формате удобном для передачи по сети в текстовом формате, в виде последовательности серии букв, которые соответсвуют передаваемым данным и по кототым можно восстановить объект. Этот плагин нужно подключать в общую часть и отдельные части JVM, JS проекта. Для серилизации существуют несколько конечных форматов XML, JSON.
Формат JSON - через запятую перечисляются поля объектов и внутренние объекты, в [ ] передаются массивы, весь файл закован в { }. Функция для серилизацию нужно 2 аргумента (серилайзер и объект, который мы серелизуем). Серилайзер - это объект, который описывает наш класс (разбивает класс на поля).
Функция stringify - преобразует объект в JSON. Функция parse - преобразует JSON в объект.
