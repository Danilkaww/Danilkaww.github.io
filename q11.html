11. Библиотека react route
Библиотека позволяет создавать пути, переключать их, а так же обновлять приложение, если был клик по ссылке, в общем случае библиотка отвечает за внешний вид и удобство приложения. hashRouter - функция, которая позволяет "обернуть" приложение и использовать функции из библиотеки react route в "обернутом" приложении. Компонент router - компонент, который позволяет использовать исторью браузера для навигации по приложению.
