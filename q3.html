3. Использование библиотеки kotlinx.html
kotlinx.html - мультиплатформенная библиотека, которая позволяет упростить работу с html в kotlin TagConsumer - интерфейс, который поглащает все написанные теги и делает из них html структуру + - функция библиотеки kotlinx.html, которая позволяет добавить текст в контент тега Плюсы: -Меньший шанс ошибки/более типобезопасный -Более легкий ввод данных -Более легкий исходный код в плане визуализации
