14. Типизированные компоненты в Kotlin react
Вид компонента высокого уровня, который позволяет использовать один и тот же компонент несколько раз, изменяя входные данные/параметры Типизированные компоненты нужны для взаимодействия с элементами заданного типа. В случае, когда нужно создать похожие компоненты из элементов разных типов, тип компонента можно оставить открытым ( или ).
